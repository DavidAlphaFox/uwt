Module Uwt.Dns

module Dns: sig .. end

libuv provides asynchronous variants of getaddrinfo and getnameinfo.

val getaddrinfo : host:string ->
       service:string ->
       getaddrinfo_option list -> Unix.addr_info list uv_result Lwt.t

Asynchronous getaddrinfo(3).

Either node or service may be NULL but not both.

Be careful: Uwt.Dns.getaddrinfo returns raw error codes, whereas Unix.getaddrinfo returns the empty list
