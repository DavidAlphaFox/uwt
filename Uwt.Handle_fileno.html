Module Uwt.Handle_fileno

module Handle_fileno: sig .. end

The usage of Uwt.Handle_fileno.fileno is unsafe and strongly discouraged. But it's sometimes necessary, if you need to interact with third parties libraries. Rules:

  • You must still keep your original handle around. Otherwise uwt will close the handle when the garbage collector removes it.
  • Always close the handle with Uwt.Handle.close (or Tcp.close, etc.), not Unix.close or any other function.
  • Never pass the file descriptor to functions like Uwt.Pipe.openpipe or Uwt.Poll.start.
